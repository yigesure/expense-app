# 高端安卓密码管理器应用

## Core Features

- 生物识别认证

- AES-256加密存储

- 智能密码生成

- 安全风险分析

- 端到端加密同步

## Tech Stack

{
  "Android": "Kotlin Multiplatform + Jetpack Compose + SQLDelight + Google Tink + AndroidX Biometric"
}

## Design

Material Design 3与Glassmorphism玻璃拟态融合风格，午夜蓝背景配青蓝渐变，圆角矩形设计配合柔光阴影和半透明效果，支持丰富的微交互动画
